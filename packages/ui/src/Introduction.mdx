## Installation

Import components from the `@repo/ui` package:

```tsx
import { Button, Card, Tabs } from '@repo/ui';
```

## Quick Start

All components support Tailwind CSS utility classes for customization:

```tsx
import { Button } from '@repo/ui';

export default function Example() {
  return (
    <Button variant="default" size="lg" className="w-full">
      Click me
    </Button>
  );
}
```

## Key Features

### Component Variants

Most components support multiple variants via the `variant` prop:

- **Button**: `default`, `destructive`, `outline`, `secondary`, `ghost`, `link`
- **Card**: Standard card with `CardHeader`, `CardContent`, `CardFooter`
- **Tabs**: Horizontal navigation with accessible keyboard controls

### Dark Mode

The library includes built-in dark mode support using CSS variables:

```tsx
import { ThemeProvider } from 'next-themes';

export default function App({ children }) {
  return (
    <ThemeProvider attribute="class" defaultTheme="system">
      {children}
    </ThemeProvider>
  );
}
```

## Component Categories

### Form Components

- **Button** - Primary action button with variants
- **Checkbox** - Accessible checkbox with label
- **Select** - Dropdown select with search

### Layout Components

- **Card** - Container with header, content, footer
- **Separator** - Visual divider
- **Tabs** - Tabbed navigation
- **ScrollArea** - Custom scrollbar styling

### Display Components

- **Avatar** - User profile image with fallback
- **Progress** - Progress indicator
- **Timeline** - Vertical timeline visualization

### Feedback Components

- **Sonner** - Toast notifications
- **Accordion** - Collapsible content sections

## Development

### Adding New Components

1. Create component in `packages/ui/src/components/ui/`
2. Export from `packages/ui/src/index.ts`
3. Create story in `*.stories.tsx`
4. Document props and usage

### Theming

Customize the design system by editing:

- `packages/ui/tailwind.config.js` - Tailwind theme tokens
- `packages/ui/src/globals.css` - CSS variables for colors
